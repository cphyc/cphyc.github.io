So you are interested in using the Chemical Equilibrium with Applications (CEA) code, but you 
don't usually work in Fortran? Well, this is your chance to use a very fast python code that 
allows you to run chemical equilibrium calculations with Python!

First of all, download the code from here: https://github.com/nespinoza/pyCEA
